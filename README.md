# Ivette Media – Website

> **Brands that stick and click**  
> Official website for Ivette Media — a Nairobi-based PR, Communication & Marketing agency.

---

## 📁 Project Structure

```
ivette-media-website/
├── index.html       # Home page
├── about.html       # About Us – founder, mission, values
├── services.html    # Services – marketing, communication, microfilms, training
├── contact.html     # Contact page – form, details, social links
├── logo.jpg         # Ivette Media brand logo
└── README.md        # This file
```

> **Note:** All CSS and JavaScript are embedded directly inside each HTML file. No build tools, no frameworks, no dependencies — just open and go.

---

## 🚀 Getting Started

### Option 1 — Open Locally
1. Download all files and keep them in the **same folder**
2. Open `index.html` in any modern web browser
3. Navigate between pages using the menu

### Option 2 — Host Online
Upload all files to any static hosting platform:
- [Netlify](https://netlify.com) — drag and drop the folder
- [GitHub Pages](https://pages.github.com) — push to a repo and enable Pages
- [Vercel](https://vercel.com) — connect your repo or drag and drop
- Any standard web hosting / cPanel via FTP

## 🌗 Dark Mode

A light/dark mode toggle (☀️ / 🌙) is built into the navigation bar on every page. The user's preference is saved to `localStorage` and persists across sessions and page navigation.

## ✏️ Customisation

### Update Contact Details
Search for and replace the following values across all HTML files:

| Placeholder | Replace with |
|------------|-------------|
| `ivettemediake@gmail.com` | Your email address |
| `+254 726 95 33 09` | Your primary phone number |
| `+254 702 33 89 03` | Your secondary phone number |
| `https://facebook.com/ivettemedia` | Your real Facebook URL |
| `https://instagram.com/ivettemedia` | Your real Instagram URL |
| `https://linkedin.com/company/ivettemedia` | Your real LinkedIn URL |
| `https://twitter.com/ivettemedia` | Your real X/Twitter URL |
| `https://youtube.com/@ivettemedia` | Your real YouTube URL |

### Update the WhatsApp Link
Find `wa.me/254726953309` in `contact.html` and replace `254726953309` with your WhatsApp number in international format (no `+` or spaces).

### Replace the Logo
Replace `logo.jpg` with your updated logo file. Keep the same filename, or update the `src` attribute in the `<img>` tags across all pages.
